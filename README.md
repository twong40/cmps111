HOW TO COMPILE AND RUN BENCHMARK

(1) cd /usr/src/
(2) Run as root (su root)
(3) make buildkernel -DKERNFAST
(4) make installkernel
(5) Restart the OS
(6) Go to benchmark directory
(7) make
(8) ./ benchmark
